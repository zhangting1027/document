| 参数  |  描述 |
|---|---|
| 标题  | 试题题目，必填  |
| 类型  | 试题类型：单选、多选、文本 ，默认为单选 |
| 标签  | 给试题添加标签  |
| 得分 |  答对获得的分数 |
|  是否启用 | 默认勾选，试题需要使用  |
|  描述   |  对试题的描述    | 
|  添加答案    | 填入试题选项和答案，必填      |
